## Installation

1. Clone the repository:
```bash
git clone https://github.com/dredog366/Initial-SecJob-Coach.git
cd Initial-SecJob-Coach
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```

4. Open your web browser and navigate to:
```
http://localhost:5000
```

## Usage

### Study Modules
Navigate to the "Study Modules" section to learn about:
- What is a SOC and SOC analyst tiers
- SIEM tools and their capabilities
- Incident response lifecycle (NIST framework)
- Common security incidents
- Threat intelligence types and MITRE ATT&CK
- Network security fundamentals and log analysis

### Take a Quiz
Test your knowledge by taking the interactive quiz:
1. Click "Take a Quiz" from the home page
2. Answer 10 multiple-choice questions
3. Get instant feedback with explanations
4. View your final score and performance

### Study with Flashcards
Review key terms and definitions:
1. Click on "Flashcards" to start
2. Click cards to flip between terms and definitions
3. Use arrow keys or buttons to navigate
4. Shuffle cards for varied practice

## Learning Topics Covered

### SOC Fundamentals
- Security Operations Center overview
- SOC analyst tier roles (Tier 1, 2, 3)
- SIEM platforms and capabilities
- Common SIEM tools (Splunk, QRadar, Sentinel, ELK)

### Incident Response
- NIST Incident Response Lifecycle
- Detection and analysis techniques
- Containment, eradication, and recovery
- Post-incident activities

### Threat Intelligence
- Types of threat intelligence (Strategic, Tactical, Operational, Technical)
- Indicators of Compromise (IOCs)
- MITRE ATT&CK Framework
- Attack lifecycle and tactics

### Network Security
- Defense in depth strategies
- Network protocols and monitoring
- Security log analysis
- Suspicious activity indicators
